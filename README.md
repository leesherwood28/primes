## Running Project

### Pre-requisits

- Have .net CLI installed
- Have npm installed

Then:

- Run the web by going to .\primes-ui and running `npm i && npm run dev`;
- Run the backend by going to .\primes-backend\primes-api and running `dotnet run`;
